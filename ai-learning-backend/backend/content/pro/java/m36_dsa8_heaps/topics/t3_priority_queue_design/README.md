# Topic 3: Priority Queue Design — Lazy Deletion, Custom Comparators

**Module**: M36 (DSA8) | **Difficulty**: ⭐⭐⭐⭐⭐⭐ (6/10)

## Key Concepts
- Lazy deletion: mark elements as removed in HashMap; skip on poll
- Task Scheduler: count frequencies; (maxFreq-1)*(n+1)+maxCount vs tasks.length
- Reorganize String: always pick highest-frequency char not equal to last placed
- Furthest Building: use min-heap for ladders — replace with bricks when possible
- K Pairs Smallest Sums: min-heap starting with (nums1[i], nums2[0]) pairs
