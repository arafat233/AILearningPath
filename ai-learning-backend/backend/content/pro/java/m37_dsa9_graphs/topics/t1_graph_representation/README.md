# Topic 1: Graph Representation — Adjacency List, BFS, DFS

**Module**: M37 (DSA9) | **Difficulty**: ⭐⭐⭐⭐⭐ (5/10)

## Key Concepts
- Adjacency list: List<List<Integer>> — O(V+E) space
- BFS: shortest path unweighted; queue + visited
- DFS: cycle detection, connected components, topological order
- Directed vs undirected: undirected adds edge both ways
- Weighted: List<List<int[]>> — each int[] is [neighbor, weight]
- Matrix: int[n][n] — O(V²) space, use for dense graphs only
- Union-Find: O(α(n)) amortized — best for connectivity queries
