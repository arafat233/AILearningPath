# Topic 2: BFS/DFS Applications — Islands, Clone Graph, Walls and Gates

**Module**: M37 (DSA9) | **Difficulty**: ⭐⭐⭐⭐⭐ (5/10)

## Key Concepts
- BFS = FIFO — level by level — shortest path in unweighted graph
- DFS = LIFO/recursion — explore depth first — cycle detection, topo order
- Islands: BFS/DFS flooding O(m*n)
- Clone Graph: DFS + HashMap for node mapping
- Walls & Gates: multi-source BFS from all gates simultaneously
- Cycle in directed: DFS with 3-state color (0=white, 1=gray, 2=black). Gray neighbor = cycle.
- Bipartite: 2-color BFS; same color neighbor = not bipartite
