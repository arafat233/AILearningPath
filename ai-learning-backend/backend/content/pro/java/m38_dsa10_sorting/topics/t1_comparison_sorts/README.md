# Topic 1: Comparison Sorts — Merge, Quick, Heap Sort

**Module**: M38 | **Difficulty**: ⭐⭐⭐⭐⭐

## Key Concepts
- Merge sort: O(n log n) O(n) — stable, best for linked lists and external sort
- Quick sort: O(n log n) avg O(n²) worst O(log n) space — unstable, cache-friendly, fastest in practice
- Heap sort: O(n log n) O(1) — unstable, in-place
- Stable = relative order of equals preserved: Merge, Counting, Radix, Tim
- Unstable: Quick sort, Heap sort
- Java Arrays.sort uses TimSort (stable, O(n log n)) for objects; dual-pivot Quicksort for primitives
