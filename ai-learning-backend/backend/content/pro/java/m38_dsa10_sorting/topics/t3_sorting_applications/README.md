# Topic 3: Sorting Applications — Intervals, Anagrams, Custom

**Module**: M38 | **Difficulty**: ⭐⭐⭐⭐⭐

## Key Concepts
- Merge intervals: sort by start; merge if prev.end >= curr.start
- Insert interval: find left/right boundaries; merge overlapping
- Largest number: custom comparator (b+a).compareTo(a+b)
- Sort by frequency: freq map + custom comparator
- Minimum platforms: sort arrivals + departures; two-pointer
- Count inversions: O(n log n) via merge sort — count when right element taken
