# Topic 5: Sort Synthesis — When to Use Which Sort

**Module**: M38 | **Difficulty**: ⭐⭐⭐⭐⭐⭐

## Key Concepts
- Array of objects, need stable: Arrays.sort (TimSort)
- Primitive array, speed: Arrays.sort (dual-pivot quicksort)
- Linked list: merge sort (no O(1) random access)
- Small range integers: counting sort
- Known d-digit base-k: radix sort
- Nearly sorted: insertion sort O(n) best case
- External sort (disk): merge sort (sequential access)
