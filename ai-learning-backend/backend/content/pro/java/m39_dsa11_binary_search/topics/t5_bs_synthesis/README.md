# Topic 5: Binary Search Synthesis — Patterns and Tradeoffs

**Module**: M39 | **Difficulty**: ⭐⭐⭐⭐⭐⭐

## Key Concepts
- Classic: target found → return; monotone 2D → split
- Search on answer: define feasibility function; binary search on [lo,hi]
- Rotated: determine sorted half then recurse
- Peak element: if a[m]>a[m+1] → peak left (hi=m)
- H-index: sort + scan with min(c[i], n-i)
- Median of two sorted: binary search on shorter array partition
