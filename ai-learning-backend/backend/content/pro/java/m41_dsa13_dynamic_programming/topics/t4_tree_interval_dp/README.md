# Topic 4: Tree DP, Interval DP — Burst Balloons, Matrix Chain

**Module**: M41 | **Difficulty**: ⭐⭐⭐⭐⭐⭐⭐⭐

## Key Concepts
- Interval DP: dp[i][j] = optimal for range [i,j]
- Burst Balloons: dp[i][j] = max coins from last balloon in [i,j]
- Matrix Chain: dp[i][j] = min multiplications for matrices i..j
- Palindrome Partitioning II: dp[i] = min cuts for s[0..i]
- Tree DP: dp on tree nodes; combine children results
- Diameter = max(leftH+rightH) across all nodes — classic tree DP
