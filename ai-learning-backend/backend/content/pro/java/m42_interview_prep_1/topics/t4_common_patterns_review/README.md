## Key Concepts
- Two pointers: sorted array, pair sum, palindrome, remove duplicates
- Sliding window: subarray/substring, variable window (shrink when invalid)
- Prefix sum: range query, subarray sum k, contiguous array
- Monotonic stack: next greater, histogram, daily temperatures
- Fast/slow: cycle detection, middle of list, palindrome LL
- Binary search on answer: Koko, split array, minimum capacity
- DP patterns: 1D (coin change, LIS), 2D (grid, edit distance, knapsack)
- Graph: BFS for shortest path, DFS for connectivity, topo for ordering
