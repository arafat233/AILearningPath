# Topic 1: STAR Method — Situation Task Action Result

**Module**: M44 | **Difficulty**: ⭐⭐⭐⭐

## Key Concepts
- STAR: Situation (context), Task (your role), Action (YOU not we), Result (measured impact)
- Metrics: 87% faster, 40%→18% abandonment, 23 min downtime — make stories credible
- 5-7 core stories that flex to cover any question
- Each story 2-3 minutes when spoken — practice with timer
- Action must be YOURS: 'I proposed', 'I implemented', 'I convinced'
- End with: what you learned and how you applied it
