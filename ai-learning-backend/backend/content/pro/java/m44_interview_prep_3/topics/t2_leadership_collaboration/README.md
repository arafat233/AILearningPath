# Topic 2: Leadership and Collaboration Stories

**Module**: M44 | **Difficulty**: ⭐⭐⭐⭐

## Key Concepts
- Leadership without title: initiative + data + empathy = strongest combination
- Conflict resolution: data before opinion, empathy before correction, propose privately first
- Collaboration: cross-functional alignment, stakeholder management, shared ownership
- Mentorship: teaching others, documentation, knowledge transfer count as impact
- Process improvement: systemic fix > one-time fix in story value
- Metrics: quantify team impact (deployment frequency, error rate, onboarding time)
