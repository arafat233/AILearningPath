# Topic 4: Career Narrative — Your Story and Motivation

**Module**: M44 | **Difficulty**: ⭐⭐⭐⭐

## Key Concepts
- 2-min self-intro: past (relevant)→present (current skills)→future (why this role)
- Strengths: 1-2 specific strengths + concrete example each (not generic)
- Weakness: real + active mitigation + evidence of improvement
- 'Why this company': specific knowledge + skill fit + long-term vision
- Questions to ask: 3 thoughtful questions about team, growth, tech stack
- Growth story: show progression and increasing scope of impact
