# Topic 1: Mock Coding Easy — 5-Minute Problems

**Module**: M45 | **Difficulty**: ⭐⭐⭐⭐

## Key Concepts
- Easy problems: Two Sum, Valid Parens, Reverse LL, FizzBuzz, Binary Search — all solvable in 5 min
- Warm-up before interview: solve 2 easy problems to get in rhythm
- Practice: set timer, solve without looking at solutions
- Always: clarify (30 sec) → approach (30 sec) → code (3 min) → test (1 min)
- Easy ≠ trivial: edge cases, integer overflow, null checks still matter
