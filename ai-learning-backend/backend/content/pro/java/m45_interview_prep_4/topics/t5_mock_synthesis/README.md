# Topic 5: Mock Round Synthesis — Full Interview Simulations

**Module**: M45 | **Difficulty**: ⭐⭐⭐⭐⭐⭐⭐

## Key Concepts
- Full mock = 45-min with coding + 10 min for questions
- Simulate realistic conditions: camera on, no IDE hints, one browser tab
- Rate yourself: code correctness, communication, time management, edge cases
- Schedule 3 full mocks/week in final 2 weeks before interview
- Debrief after each mock: 3 wins + 3 areas to improve
- Key metric: can you solve a medium in 15 min with clear communication?
